# Small Web Page Projects with JavaScript

JavaScript practice projects (vanilla HTML/CSS/JS, no frameworks or build tools), including several exercises and mini-apps from a structured JavaScript course. Each project lives in its own folder — open its `index.html` in a browser to run it.

## Projects

| Project | Description |
|---|---|
| [Bankist App - Stage 1](./Bankist%20App%20-%20Stage%201) | Early-stage build of the "Bankist" fake banking app UI/UX project |
| [Bankist App - Stage 2](./Bankist%20App%20-%20Stage%202) | Mid-stage build of the Bankist project, expanded script logic |
| [Bankist App - Stage 3](./Bankist%20App%20-%20Stage%203) | Most complete Bankist build, with full image assets and expanded styling/script |
| [Developer Skills](./Developer%20Skills) | Early exercise practicing core JS developer workflow/skills |
| [Fundamentals](./Fundamentals) | JavaScript fundamentals practice (variables, operators, control flow, etc.) |
| [HTML-CSS](./HTML-CSS) | HTML/CSS-only warm-up exercise (no JavaScript) |
| [Guess my Number](./Guess%20my%20Number) | Classic number-guessing game with score/high-score tracking |
| [Pig Game](./Pig%20Game) | Two-player dice game with turn-based scoring and "hold"/"win" logic |
| [Modal](./Modal) | Reusable modal/popup component with open/close and keyboard (Esc) handling |

*The three Bankist folders are progressive checkpoints of the same project saved at different build stages, not separate apps — kept to show progression rather than only the final result.*

## Tooling

Most folders include a `.prettierrc` for consistent code formatting via [Prettier](https://prettier.io/).
